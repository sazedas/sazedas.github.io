RobotKid UX/UI Design

Overview

RobotKid, a new company that aims to help kids who cannot attend school be part of school as much as possible. RobotKid is a software company, and in particular wants to leverage existing telerobots to provide an outstanding experience that goes beyond ‘just being there’. The company has sought me and my team out, as designers. All of the software design was in our hands, as RobotKid had the idea protected (meaning no competition), but has no idea how to actually design the software. We used our own ideas including ideas from other sources to create the design of the RobotKid's user interface for children.

Detailed Description of Individual Mockups

User Interface Screen 1


Icon

Description


Mirrored Video Feed (User-Facing): Displays live recording of the user who is using RobotKid remotely.


Menu Button (Untoggled): Button that displays a drop-down menu when clicked.


Clear Mode Button: Button indicating whether Clear Mode is on. When the background is gray, Clear Mode is turned off.


Speaker Mode Button: Indicates whether the user’s microphone is active or muted.


Volume Control Slider: Slider that user can use to control the volume of the audio.


System Status Indicators: Informative icons that describe the status of the system. In order from left to right: battery indicator displaying how much battery RobotKid has, icon displaying the current WiFi signal strength, and current time and date.


Lecture Mode Recording Icon: States whether lecture mode is on. Top icon signals that the current video feed from the school is being recorded for offline use. Bottom button slider.


Lecture Mode Toggle Switch (Toggled): Switches Lecture Mode from on to off. Button is switched to off.


Call for Attention Button: Toggles RobotKid’s LED light to turn on.





User Interface Screen 2


Icon

Description


Menu Button (Toggle): Button that displays a drop-down menu when clicked. The Menu has been clicked and is showing the drop-down button.


Clear Mode Button (Engaged): Button indicating whether Clear Mode is on. When the background is green, Clear Mode is turned on.


Settings Button: Displays various options and preferences, including brightness, connection status, etc


Games Button (Expanded): Provides four games that can be played between the user and another person (who is present with the RobotKid).


Color Button: Opens up color customization options for the UI.


Accessibility: Displays various accessibility features including smartboard integration (for notetaking) and integration with Microsoft’s Adaptive Xbox Controller to allow children of varying abilities to use the RobotKid.


Lecture Mode Recording Icon: States whether lecture mode is on. Top icon signals that the current video feed from the school is being recorded for offline use. Bottom button slider.


Help Button: Provides the user with a phone number for technical difficulties as well as the option to open a live chat client to directly discuss with the school’s IT department.
